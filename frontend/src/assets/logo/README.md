# Logo 文件目录

请将 Logo 文件放在此目录下。

## 文件要求

**推荐格式**：
- **SVG**（优先）：矢量图，可缩放，文件小
- **PNG**：如果只有PNG，请提供透明背景版本

**文件命名**：
- `logo.svg` 或 `logo.png`（主Logo）
- 如果有深色/浅色版本：`logo-light.svg`、`logo-dark.svg`

**尺寸建议**：
- SVG：任意尺寸（会自动缩放）
- PNG：建议提供 @2x 版本（如 200x60px 的Logo，提供 400x120px）

## 使用方式

将Logo文件放在此目录后，我会更新 `frontend/src/components/Logo.vue` 组件来使用真实的Logo文件。
