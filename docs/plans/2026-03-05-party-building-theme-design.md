# 党建主题样式统一化设计方案

**日期:** 2026-03-05
**设计者:** Claude
**状态:** 已批准

---

## 一、需求概述

将现有的党建主题样式（以 Header 为代表）统一应用到所有组件，实现整体视觉风格的统一性。

### 当前状态
- ✅ `party-theme.css` 已定义党建色彩系统
- ✅ Header 组件已应用党建红色渐变背景
- ❌ 其他组件仍使用默认蓝色主题

### 目标状态
- 所有导航/布局组件使用党建主题
- 所有自定义业务组件使用党建主题
- 保持视觉一致性：中国红 + 金色点缀 + 庄重风格

---

## 二、设计原则

### 2.1 色彩系统
- **主色系:** 中国红渐变 `#C8102E → #E84D56 → #8B0000`
- **辅色:** 金色 `#FFD700`（装饰、强调）
- **背景:** 白色 `#FFFFFF`（卡片、容器）
- **文字:** 深灰 `#1A1A1A`（正文）、白色 `#FFFFFF`（红色背景上）

### 2.2 形状与边框
- **圆角:** 减小圆角（4px-8px），增强庄重感
- **边框:** 使用金色装饰线（左侧、顶部、底部）
- **阴影:** 使用淡红色阴影 `rgba(200, 16, 46, 0.08)`

### 2.3 装饰元素
- **金色左边框:** 卡片容器标识（3px）
- **金色底线:** 标题装饰（2px）
- **红色渐变:** 主要按钮、徽章

---

## 三、实施方案：混合方案

### 3.1 全局 CSS 类（通用模式）

在 `party-theme.css` 中新增可复用类：

**容器类**
- `.party-card`: 白底 + 金色左边框
- `.party-section`: 内容分组容器

**标题装饰类**
- `.party-title-underline`: 金色底线
- `.party-title-bar`: 左侧金色装饰条

**按钮类**
- `.party-btn-primary`: 红色渐变主按钮
- `.party-btn-secondary`: 金色边框次按钮

**徽章/标签类**
- `.party-badge`: 金色徽章
- `.party-tag`: 红色标签

**分割线类**
- `.party-divider`: 金色渐变分割线
- `.party-divider-red`: 红色分割线

**强调文字类**
- `.party-text-gold`: 金色强调文字
- `.party-text-red`: 红色强调文字

### 3.2 Tailwind 配置扩展

扩展 `tailwind.config.js`：

```javascript
colors: {
  // 党建红色
  'party-red': {
    DEFAULT: '#C8102E',
    light: '#E84D56',
    dark: '#8B0000',
  },
  // 党建金色
  'party-gold': {
    DEFAULT: '#FFD700',
    light: '#FFE55C',
    dark: '#CCB800',
  },
  // 党建深红
  'party-deep-red': '#8B0000',
}
```

### 3.3 组件改造计划

#### 阶段一：导航/布局组件（优先级高）

| 组件 | 文件 | 改造内容 |
|------|------|----------|
| Logo | `Logo.vue` | 添加金色党徽图标 |
| ModuleSwitcher | `ModuleSwitcher.vue` | 激活状态使用红色渐变 |
| SidebarMenu | `SidebarMenu.vue` | 使用红色渐变背景 |
| MenuItem | `MenuItem.vue` | 激活项金色左边框 |

#### 阶段二：自定义业务组件（优先级中）

| 组件类型 | 代表组件 | 改造内容 |
|----------|----------|----------|
| 卡片容器 | AgentList, ConversationList | 应用 `.party-card` 类 |
| 聊天界面 | ChatInterface, MediaChatInterface | 消息气泡红色主题 |
| 输入区域 | ChatInput, MediaInput | 发送按钮红色渐变 |
| 预览面板 | PreviewPanel | 工具栏金色分割线 |

#### 阶段三：页面级优化（优先级低）

| 页面 | 文件 | 优化内容 |
|------|------|----------|
| 登录页 | `LoginPage.vue` | 登录按钮红色渐变 |
| 管理页面 | AdminLayout | 统一党建主题 |

## 四、视觉规范

### 4.1 卡片容器
```
┌─────────────────────────┐
│ ███ 标题                 │ ← 金色左边框 3px
├─────────────────────────┤
│                         │
│  内容区域               │
│                         │
└─────────────────────────┘
```

### 4.2 标题装饰
```
标题内容
════════                    ← 金色底线 2px
```

### 4.3 按钮
- **主按钮:** 红色渐变 + 白色文字 + 4px 圆角
- **次按钮:** 白底 + 金色边框 + 深红文字

### 4.4 徽章
- **金色徽章:** 金黄渐变背景 + 深红文字
- **红色标签:** 红色渐变背景 + 白色文字

## 五、技术实现要点

### 5.1 样式优先级
1. 优先使用 Tailwind 类（如 `bg-party-red`）
2. 复杂样式使用全局 CSS 类（如 `.party-card`）
3. 特殊需求使用内联样式

### 5.2 兼容性
- 保持 Element Plus 组件功能不变
- 仅覆盖视觉样式
- 不影响现有交互逻辑

### 5.3 响应式
- 移动端保持相同的色彩系统
- 装饰线尺寸在小屏幕上适当缩小
