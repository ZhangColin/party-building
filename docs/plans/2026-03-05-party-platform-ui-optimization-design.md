# 党建AI智能平台 UI优化设计文档

**日期**: 2026-03-05
**目标**: 优化导航结构和AI对话界面党建风格

---

## 一、设计目标

1. **导航结构清晰化**：前台展示AI工具，后台专注管理功能
2. **党建风格统一**：AI对话界面左侧区域应用党建主题色
3. **用户体验优化**：简化前台导航，强化后台管理

---

## 二、设计方案

### 2.1 导航结构调整

#### 前台导航（顶部ModuleSwitcher）
**保留模块**：
- AI党建助手
- 知识库

**移除模块**：
- ~~党员管理~~ → 移至后台
- ~~组织生活~~ → 移至后台
- ~~党费管理~~ → 移至后台

#### 后台导航（AdminLayout侧边栏）
**现有菜单**：
- 用户管理

**新增菜单**：
- 党员管理（路径：`/admin/party-members`）
- 组织生活（路径：`/admin/organization-life`）
- 党费管理（路径：`/admin/party-fees`）

**图标配置**：
- 党员管理：`user-group`
- 组织生活：`calendar`
- 党费管理：`currency-yen`

---

### 2.2 AI对话界面党建风格改造

#### 改造范围
**需要调整的元素**：
- ✅ 左侧工具列表（ToolList组件）
  - 工具项选中状态：党建红 #C8102E
  - 工具分类标题：红色文字
  - 悬停效果：浅红背景

- ✅ 新建对话按钮
  - 背景色：党建红渐变 `linear-gradient(135deg, #C8102E 0%, #E84D56 100%)`
  - 文字：白色
  - 边框：可选金色装饰 `1px solid rgba(255, 215, 0, 0.3)`

- ✅ 用户头像区域
  - 默认头像：党徽图标或红色圆形占位符
  - 边框：金色点缀

**不需要调整**：
- ❌ 消息气泡（UserMessage、AssistantMessage）
- ❌ 聊天整体布局
- ❌ 输入框区域（保持Element Plus默认样式）

#### 配色方案
```css
/* 党建主题色 */
--party-primary: #C8102E;      /* 中国红 */
--party-secondary: #E84D56;    /* 浅红 */
--party-deep-red: #8B0000;      /* 深红 */
--party-gold: #FFD700;          /* 金色 */
--party-gold-light: rgba(255, 215, 0, 0.3);
```

## 三、技术实施

### 3.1 涉及文件

**配置文件**：
- `configs/navigation.yaml` - 移除三个管理模块

**前端组件**：
- `frontend/src/layouts/AdminLayout.vue` - 添加侧边栏菜单
- `frontend/src/components/ModuleSwitcher.vue` - 只显示2个前台模块
- `frontend/src/components/ToolList.vue` - 工具列表党建风格
- `frontend/src/components/UserAvatar.vue` - 用户头像样式

### 3.2 实施步骤

1. **更新导航配置**
   - 从 `navigation.yaml` 移除三个 page 类型模块
   - 确保只保留 toolset 类型的AI党建助手和知识库

2. **改造后台布局**
   - 在 AdminLayout 侧边栏添加三个菜单项
   - 配置路由和图标
   - 添加点击事件处理

3. **AI对话界面党建化**
   - 修改 ToolList 组件样式
   - 调整新建对话按钮样式
   - 优化用户头像显示

4. **测试验证**
   - 前台导航只显示2个模块
   - 后台侧边栏显示4个菜单项
   - AI对话界面党建风格统一
